### Host Locally or on VPS

```sh
$ git clone https://github.com/KazeDevID/Tele-chan
$ cd Tele-chan
```

* Rename `.env.example` to `.env` and fill it up with your `TOKEN`

```sh
$ pip3 install -r requirements.txt
$ python3 waifu.py
```
Enjoy!!
